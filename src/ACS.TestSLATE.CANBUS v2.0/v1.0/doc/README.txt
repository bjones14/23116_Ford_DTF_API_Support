This folder contains the plugin documentation.
It contains three subfolders
\designs
\requirements
\test plans

All Designs and Modification Plans go in the \designs folder.
All Requirements documents go in the \requirements folder.
All Test Plans go in the \test plans folder.

This plugin contains the TestSLATE Plugin HOWTO.docx in the \design folder as this document forms the basis for Test SLATE Plugins.

There is a sample Test Plan in the \test plans folder.

The XML file generated during the build process should be stored in the \doc folder. This file is a programmer's API reference to the dll. Any other programmer references should also be placed in this folder.

The Plugin Template Programmer Reference.docx is located in this folder.
